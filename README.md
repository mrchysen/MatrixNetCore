# MatrixNetCore

MatrixNetCore - набор классов для численных методов на .net

## Функционал

* Вектора и операции над ними
* Матрицы NxM и операции над ними
* Полиномы
* Методы решения СЛАУ итерационные / прямые
* Методы нахождения собственных чисел / векторов
* Методы нахождения значения интеграла
* Методы решения не линнейных уравнений
* Методы решенеия нелинейных систем

* Вектора и матрицы можно сохранять в файл
